第 $012$ 篇题解。

Main Idea

给你 $n$ 个数，求那个出现了奇数次的数是哪个。

Analysis

由于 $1 \leq n \leq 10^7 + 1$ 且 $1 \leq a_i \geq 10^9$，我们必须在 $\mathit{O}(n)$ 的时间复杂度里完成计算。

此题使用位运算比较合适（快）。

c++ 中，按位异或（^）运算：相应位上的值相同，就返回 $0$，否则返回 $1$（当然是二进制中）。如 09 和 11 异或的结果是 02。详细运算如下：
