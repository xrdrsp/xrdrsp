Posts

Comment

This blog uses Disqus to show comments. To leave a reply, you should log in with Disqus, Twitter, Facebook or Google. If you are in China, Disqus probably could not be displayed.

If you do not have any of the website accounts above, or you cannot visit Disqus, it is recommended to comment via Issue s. Please check if there are Issue s to the post. If not so, please create an issue entitled [Comment] $post-name.

Report

If anything unexpected happened to you on this website, you can create an Issue entitled [Bug-Report] $title.

If you have ideas about the blog, you can create an Issue entitled [Feature-Request] $title.
